# Sistema de Gestión Escuela de Música

## Breve descripción
Este proyecto consiste en el desarrollo de una aplicación web para la gestión interna de una escuela de música. 
La plataforma integra funcionalidades de administración, comunicación, gestión académica y control económico, adaptadas a los distintos roles del sistema: administrador, profesor, alumno y visitante.

El objetivo principal es digitalizar los procesos habituales de la escuela y centralizar la información de alumnos, profesores, cursos, matrículas, pagos y eventos, facilitando la gestión diaria del centro y mejorando la comunicación interna.


## Funcionalidades principales

- Gestión de usuarios, cursos y matrículas  
- Control del estado de los pagos  
- Publicación de anuncios y recursos digitales  
- Consulta pública de eventos y fechas importantes 

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- JQuery
- PHP
- MySQL
- Git y GitHub
