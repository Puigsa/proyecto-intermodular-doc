# Sistema de Gestión Escuela de Música

## Breve descripción
Este proyecto consiste en la creación de un **sistema de gestión digital** para una escuela de música de grado elemental.  
Permite gestionar cursos, alumnos, profesores, calendario de eventos, calificaciones y comunicación interna, además de ofrecer sistemas de denuncia anónima y reserva de aulas.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Git y GitHub
